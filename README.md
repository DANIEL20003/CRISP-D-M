# Proyecto de Minería de Datos - CRISP-DM

Este repositorio contiene un proyecto de minería de datos estructurado siguiendo la metodología CRISP-DM.

## Estructura del Repositorio

- **/1_business_understanding**: Documentos relacionados con la comprensión del negocio y los objetivos del proyecto.
- **/2_data_understanding**: Notebooks y scripts para la exploración y análisis inicial de los datos.
- **/3_data_preparation**: Scripts y notebooks para la limpieza, transformación y preparación de los datos.
- **/4_modeling**: Código y notebooks para la construcción y entrenamiento de modelos.
- **/5_evaluation**: Scripts y notebooks para la evaluación de los modelos.
- **/6_deployment**: Archivos y scripts necesarios para el despliegue del modelo.
- **/data**: Contiene los datos del proyecto.
  - **/data/raw**: Datos crudos y sin procesar.
  - **/data/processed**: Datos procesados y listos para el modelado.
- **/models**: Modelos entrenados y guardados.
- **/reports**: Informes, visualizaciones y resultados generados.
- **/src**: Código fuente modularizado.
