# Fase 5: Evaluación

En esta fase, se evaluarán los resultados del modelo para determinar si se han cumplido los criterios de éxito de la minería de datos y cómo estos resultados contribuyen a los objetivos de negocio.

## Objetivos Clave (para la Segmentación RFM):
1.  **Analizar los Segmentos:** Calcular las características promedio (Recencia, Frecuencia, Gasto Monetario) de cada segmento para entender su perfil.
2.  **Visualizar los Segmentos:** Crear gráficos (ej. diagramas de caja, nubes de puntos 3D) para visualizar las diferencias entre los segmentos.
3.  **Interpretar y Nombrar los Segmentos:** Dar nombres descriptivos y accionables a cada segmento (ej. "Clientes Campeones", "Clientes Leales", "En Riesgo de Abandono", "Nuevos").
4.  **Validar Criterios de Éxito:** Comprobar si se alcanzaron los criterios técnicos definidos en el `project_charter.md` (ej. Coeficiente de Silueta > 0.5).
5.  **Proponer Estrategias de Negocio:** Sugerir acciones de marketing específicas para cada segmento, alineadas con los objetivos de negocio.
